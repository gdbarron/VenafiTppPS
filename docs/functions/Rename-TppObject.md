# Rename-TppObject

## SYNOPSIS
Rename an object of any type

## SYNTAX

```
Rename-TppObject [-Path] <String> [-NewName] <String> [[-TppSession] <TppSession>] [<CommonParameters>]
```

## DESCRIPTION
Rename an object of any type

## EXAMPLES

### EXAMPLE 1
```
Rename-TppObject -Path '\VED\Policy\My Devices\OldDeviceName' -NewName 'NewDeviceName'
```

Rename device
