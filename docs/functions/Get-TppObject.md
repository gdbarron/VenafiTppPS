# Get-TppObject

## SYNOPSIS
Get object by path

## SYNTAX

```
Get-TppObject [-Path] <String> [[-TppSession] <TppSession>] [<CommonParameters>]
```

## DESCRIPTION
Get object by path

## EXAMPLES

### EXAMPLE 1
```
Get-TppObject -Path '\VED\Policy\My object'
```

Get an object by full path

## PARAMETERS

### -Path
